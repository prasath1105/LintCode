### 352. PrettyPrint

#### Description
```
给出文本 text ，其中有 n 条句子，每个句子由数个单词组成（数量大于 0），以及一个宽度 width，
请将句子进行美观打印在宽度为 width 的窗口内。
打印的规则如下：

1 两条句子不应该在同一行，即打印完一个句子后应该换行。
2 每个句子中，单词应该从左往右顶格打印，两个单词之间用一个空格隔开。
3 如果一行中右侧剩下的空格不足填入新的单词，那么换行，从头开始打印。
4 每一行末尾要用空格填充，使其长度和 width 相等。
5 打印结果要用一圈星号 '*' 包装，星号不计入宽度中。

给定的文本中，单词的数量为 m，1≤m≤200，
每个单词的长度为 l，1≤l≤20。
宽度为 width，2≤width≤80，保证 widthwidth 大于等于最长的单词长度。
```

#### 思路：
```
模拟每个段落打印的过程。对段落的单词进行遍历：
1. 首先打印第一个单词"words[0]"（题目保证单个单词一定不会超过width）
2. 尝试打印剩下的"_"+"words[i]"，在不超过width的前提下能打多少是多少。最后用"_"补齐该行的width。将该行字符串加入rets。
3. 如果该段落没有打完，则返回1处理该段落余下的单词。
4. 如果该段落已经打完，则返回1处理其他段落。
5. 包裹星号
```
